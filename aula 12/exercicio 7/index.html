Capítulo 1: Estrutura básica do HTML

O HTML é a linguagem de marcação usada para estruturar conteúdo na web. Todo documento HTML começa com a declaração <!DOCTYPE html> e possui elementos como <html>, <head> e <body>.

Capítulo 2: Cabeçalhos e parágrafos

Os cabeçalhos (<h1> a <h6>) definem títulos e subtítulos, enquanto o <p> define parágrafos de texto. É importante usar corretamente os níveis de cabeçalho para organizar o conteúdo.

Capítulo 3: Links e imagens

Links são criados com o elemento <a> e permitem navegar entre páginas. Imagens podem ser inseridas com <img>, sempre incluindo o atributo alt para acessibilidade.

Capítulo 4: Listas e seções

Listas ordenadas (<ol>) e não ordenadas (<ul>) organizam informações. O elemento <section> ajuda a dividir o conteúdo em partes lógicas dentro do artigo.
