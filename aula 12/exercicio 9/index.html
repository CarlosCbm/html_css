FAQ - Perguntas Frequentes

Encontre respostas para as dúvidas mais comuns.

1. O que é HTML?

HTML é a linguagem de marcação usada para estruturar conteúdo na web, criando páginas e definindo elementos como textos, imagens e links.

2. Para que serve o CSS?

O CSS é usado para estilizar páginas HTML, definindo cores, fontes, tamanhos, layouts e outros aspectos visuais dos elementos.

3. Qual a diferença entre <section> e <article>?

<section> é usado para dividir conteúdos em blocos temáticos, enquanto <article> representa um conteúdo independente que pode ser republicado, como um post ou notícia.

4. O que é um link âncora (#)?

Um link âncora permite navegar para uma seção específica dentro da mesma página usando um ID do elemento.

5. Como tornar uma página responsiva?

Para tornar uma página responsiva, usamos CSS com media queries, layouts flexíveis (flexbox ou grid) e imagens que se ajustam ao tamanho da tela.
